# 02 META 系统（可复用框架）

> 本文档定义 Meta 元系统。**Meta 独立于具体 Core 玩法,是可复用框架**——同一套 Meta 未来可驱动合成、消除等不同 Core。
> Meta 与 Core 的连接只通过「资源流」：Core 产出分数/资源 → Meta；Meta 提供能力/加成 → Core。技术接口见 05。
>
> 设计依据请对照 `01_设计哲学.md`。本文档只写「是什么、怎么做」。

---

## Meta 总览

> **主题与角色解耦**:消除元素是魔法道具;角色是独立的**二次元萌宠精灵**(圆润毛绒、靠职业道具差异化,见 `06`),承载技能+铭文+收集+抽卡。

Meta 以**萌宠角色**为核心,由这些子系统组成:

| 子系统 | 类型 | 作用 | 对应玩家需求 |
|---|---|---|---|
| 萌宠角色(12只) | 质变·主动一次(2只被动) | 一个技能改变玩法,一局带一个;可升级 | 难度匹配 / high ceiling |
| 铭文(9格碎片制) | 量变·被动持续 | 9格碎片分配,只量变;王者式可挂单只/全局 | 确定性成长(养成) |
| 抽卡系统 | 获取 | 靠高分解锁,获得萌宠(首抽必出借贷) | 确定性成长(实力驱动) |
| 皮肤系统 | 外观 | 纯审美,变现 | To be somebody |
| 成就/排行榜 | 社交 | 奖杯+排名+称号,被看见 | To be somebody |

**核心循环**:玩消除(魔法道具)→ 打高分 → 高分给抽卡机会 + 铭文碎片 → 抽萌宠(首抽借贷,后续随机)+ 养铭文(9格碎片)+ 升级萌宠 → 用萌宠技能(一局一次,第二次看广告)+ 铭文量变加成 → 玩更高分 → 上周赛/刷成就被看见。

## 一、角色系统（核心 · 二次元萌宠）★

> **主题与角色解耦**:消除元素是**魔法道具**(见 `04`/`06`);角色是**独立的二次元萌宠精灵**(圆润毛绒小生物,靠"职业道具+属性色"差异化,见 `06`),承载技能 + 收集 + 抽卡 + 铭文养成。

**定位**:每个萌宠 = 一个二次元精灵形象 + 一个技能 + 可配的铭文(见二)。

**三条铁律**:
1. **一局只能上一个萌宠**(强制流派选择)。
2. **主动技能一局只用一次**(强力但不填平难度——一次性王牌,非持续开挂;靠「一局一次」守住难度,见 `01` 两条铁律)。被动型萌宠例外(整局生效,见下表标注)。
3. **主动技能第一次免费,第二次起看广告续用,有上限(1-2 次)**——超上限不能再续(否则破坏「一局一次」难度基石)。这是「广告」变现腿落点:卖机会非强度。

**生死线**:**锦上添花非雪中送炭**——没带任何萌宠,光裸 Core 也能正常玩、能过(守 low floor,也是 `09` 算法评估基准)。

**萌宠可升级(养成深度)**:萌宠可升级,**技能效果随等级成长**(低级保守、高级强)——但**始终受「一局一次/被动持续」的约束不变**。如破障萌宠 1 级破 1 个、高级破 2-3 个。升级让王牌更强,但还是一局一张牌(量变成长,不解锁新的填平难度的能力)。升级资源靠通关高分/碎片(实力驱动)。

### 1.1 首发萌宠全表（12 只 = 1 默认 + 11 可玩）

| # | 萌宠 | 技能效果 | 类型 | 等级成长 |
|---|---|---|---|---|
| 0 | **默认精灵**(幸运吉祥物) | 无战斗技能;游戏门面 + **对局中给基础提示**(高亮可走的步,引路降挫败) | 常驻门面·非可玩 | 提示克制(卡住才提示) |
| 1 | **借贷** ★核心·首抽必得 | 借一个特效(4连直线/T·L爆炸/5连彩球效果),**本关内必须还**(让某特效降格普通棋子,玩家自选),**不还不算过关** | 质变·一局一次 | 高级可借更强特效 |
| 2 | **时间回退** | 回退最近 5 步 | 质变·一局一次 | 高级回退更多步 |
| 3 | **存档快照** | 存一个局面,可一键跳回 | 质变·一局一次 | 高级可多存 |
| 4 | **隔位对换** | 正常只能换相邻1步,这个能换相邻2步(隔一个) | 质变·一局一次 | — |
| 5 | **重力翻转** | 翻转盘面重力一次,全部反向下落重排 | 质变·一局一次 | — |
| 6 | **彩球护盾** | 这局保彩球一次(被碰只掉护盾,彩球保留) | 质变·一局一次 | 高级保更多次 |
| 7 | **同类消除** ★厉害款 | 选一种棋子,消除全场所有该种类 | 质变·一局一次 | 高级额外触发连锁/给特效 |
| 8 | **预知** | 高亮接下来**最优的几步走法**(基于当前盘面,**不剧透掉落**)——**调用 `09` 求解器算最优路径** | 质变·一局一次 | 低级亮1步,高级亮3-5步 |
| 9 | **破障** | 直接清除场上障碍(冰/锁等) | 质变·一局一次 | **破几个看等级**:1级破1个→高级破2-3个 |
| 10 | **连消奖步** | 打连锁时奖励步数,递进式(等级越高所需连锁越低:连锁8奖1步→连锁5奖1步);**始终靠"打连锁"技巧才给,不白送** | **被动型·整局生效** | 高级所需连锁更低 |
| 11 | **连击收集** | 打连击时**额外收集铭文碎片**(养成加速;奖励"打连击"=核心爽点,正向激励,天然防刷) | **被动型·整局生效** | 高级收集更多 |

> **两个"被动型萌宠"(#10/#11)**是新类别:不是"一局一次的瞬发技能",而是**整局持续生效的被动**——带上它整局有效。代价仍是"一局只带一个"(带它就不能带战斗技能),取舍成立。
> **预知(#8) = 求解器的玩家化应用**:`09` 的求解器(后台算关卡难度)在这里变成玩家可见的功能(算最优步骤给玩家看)——算法从后台评估工具,变成可抽可用的酷技能。

### 1.2 借贷技能（核心，等价交换化身，你偏爱的"基础但上限高"）
- **借**:立刻获得一个特效(4连直线/T·L爆炸/5连彩球 的效果)。
- **还**:**本关内必须还**——还=让你某个特效失效降格成普通棋子(**玩家自选**牺牲哪个,策略:还不重要的那个)。
- **罚**:**关末若有未还的债 → 不算过关**(硬约束,逼玩家规划借与还)。
- **必须还是借贷的灵魂**(等价交换)——**"不用还"会摧毁这个技能的身份,绝不做**;要白嫖特效应是别的技能。
- 完整 LFHC:新手简单借个特效还得起;高手精算「借什么、何时借、还哪个最优」。基础好用、上限高,强度受还债制约。

> 角色 = 二次元萌宠 + 能力载体,**非叙事载体**(不做剧情)。靠美术(萌宠精灵,见 `06`)做辨识度 + 一句话能力标签。后续 LiveOps 持续出新萌宠(候选:斜向消、控制下落、跨区消、抓取暂存、连引等,均按"一局一次"或"被动"设计)。
> **合并/变身类技能**:讨论过但未敲定(合并升级在消除盘面里规则未理顺,变身术易填平难度),首发不做,留作后续。

## 二、铭文系统（量变层 · 9 格碎片制 · 不分线 · 王者式双模绑定）

> **铁律(见 `01`)**:**铭文只能量变,不能质变**——只调数值(多一点/强一点),不给新能力、不改规则、不替玩家做事。给新能力/改规则的是"角色技能",不是铭文。
> **不填平难度靠"格子有限 + 取舍"**:9 格,装了这个就装不了那个,机会成本就是代价。

### 2.1 核心机制:9 格 + 碎片制
- **每个萌宠/或全局有 9 个铭文格子**,每格放一个铭文。
- **铭文是"碎片"——单个效果极小,要堆够数量才出明显效果**(如 +步数每个+0.33,3个才=1步)。
- **9 格自由分配**:想要某效果明显就得堆多格 → 放弃其他。**全押一种也行,但不一定划算**(放弃了所有别的)。
- **没有核心格子、没有额外限制**——全靠"9 格怎么分"这一个取舍。碎片制+机会成本天然防碾压。

### 2.2 五种铭文（不分线，从这 5 种里选着填 9 格）

| 铭文 | 单个效果 | 凑整 / 上限 |
|---|---|---|
| **+步数** | 每个 +0.33 步 | 3个=+1步,**6个=+2步(封顶,>6无效)** |
| **+积分** | 每个 +X% 分数 | 线性,无封顶(冲分/换资源) |
| **+金币** | 每个 +X% 金币 | 线性,无封顶 |
| **+强化技能次数** | — | **6格 = 技能多用1次**(高成本,占2/3格) |
| **+开局奖励** | — | **3格 = 开局1个"4消直线特效";6格 = 升级成"5消食彩球"** |

**自然结论(取舍很干净)**:+2步/+1次技能/开局食彩球 **都要6格** → 几个强效果**互斥**(9格装不下两个6格组合)→ 玩家只能选一个主攻 + 3格辅助。
- 平民配法:3+3+3(如 +1步、一些积分、一些金币)。
- 极端配法:6+3(一个强效果+一点辅助)或 9 押一种。

### 2.3 绑定方式:王者式双模（玩家选，挂单只 or 全局）
和王者荣耀一样,玩家可以选这套铭文页**怎么用**:
- **挂在单个萌宠上**:这只萌宠专用的铭文页(为它定制)。
- **设为全局**:所有萌宠共用这套铭文页(省心通用)。
- **玩家自己决定**——想为某只主力萌宠定制就挂单只;想一套走天下就设全局。可配多套页,自由切换。

### 2.4 获取与边界
- **铭文/碎片靠"通关分越高得越多"获得+升级**(实力驱动);**连击收集萌宠(#11)可额外加速**。**绝不卖钱**(否则变氪金堆数值,违背原则)。
- **量变铁律**:任何铭文都不能"给新能力/改规则/替玩家做事"(那是质变=技能)。判断:点了铭文是"做同样的事多拿点(量变,✅)"还是"能做新的事/难度被抹平(质变,❌砍)"。
> (注:数值向——碎片产出要够养、别太肝,尤其欧美玩家肝度容忍低;原型/运营调。)

## 三、抽卡系统（获取层）

**定位**：角色通过抽卡获得,**抽卡机会靠「把关卡分玩得很高」解锁**(实力驱动,非花钱/等时间)。

**首抽保底**：**初始抽卡 100% 出借贷**(如阴阳师首抽必出雪女)——所有新玩家从同一个好用的核心技能起步,新手体验可控,且自然形成「借贷为主力」(90% 玩家用借贷,10% 用抽到的特色技能)。

**后续随机**：首抽之后,抽卡为随机池(其他技能),靠高分解锁机会。

**与原则一致**:靠实力解锁的抽卡不违背「不信廉价成长」;借贷保底「人人能玩」(雪中送炭由它兜底),抽到的其他技能是「锦上添花」→ 抽卡不必需,玩家不焦虑、不逼氪。

**注意**:抽卡给「角色(玩法能力)」;外观(皮肤)是另一条线(见四)。能力靠抽卡,外观靠买,严格分开。

---

## 四、皮肤系统（外观/变现层）

**定位**：纯外观,不带任何数值/能力加成。变现的干净第三条腿。

**设计原则**：
- **绝不带数值/能力加成（铁律）**：一旦沾强度,就破坏公平、违背原则。皮肤管外观,角色管能力,严格分开。
- 「换皮肤仍可识别」：物种（类别）+ 进化阶段（等级）靠核心轮廓识别,皮肤只换表面装饰,不改可识别结构。一只猫无论什么皮肤,仍一眼认出是猫、是成体。
- 无限可做、零平衡负担：节日皮、联动皮、稀有皮,持续出,只需美术（可外包）。

**作用**：
- 喂「to be somebody」：外显身份和炫耀（高手顶稀有皮肤的动物上排行榜）。
- 长期变现 + 持续新鲜感来源。

## 五、排行榜 / 竞技系统

**定位**：承接 high ceiling,让「玩得好」被看见。

**设计**：
- **同题竞技**：同一个关卡/同一局面,所有人比谁的解更优（更高分/更少步/更高级组合）。把 Core 的「最优解空间」变成可比的竞技。
- 这天然契合算法（同一道题比谁解得好,本质是把竞赛搬进休闲游戏）。
- 前提：Core 必须有「客观优劣」（分数/步数可比）——合成、消除都满足。
- 配套：完美通关标记、好友分数对比、稀有皮肤展示——让高手的炫技有舞台。

**为什么重要**：To be somebody 是三大需求之一,而多数休闲游戏这块做得弱——这是本项目可强化的差异点。

## 六、Meta 与 Core 的连接（接口概览）

**只通过资源流连接,不依赖 Core 的具体玩法：**
- Core → Meta：Core 产出标准化的「分数」和「资源」。
- Meta → Core：Meta 提供「角色能力（一个）」和「铭文加成（一组数值）」给 Core 生效。

**这意味着**：Meta 不关心 Core 是合成还是消除,只关心「收到多少分/资源」和「要把哪个角色能力、哪些铭文加成注入当前局」。技术实现见 `05_技术架构.md`。

**防割裂**：Meta 与 Core 机制可以无关（如 Gossip Harbor 的合成与庄园）,但要靠「一致的动物主题包装 + 顺畅的资源流」让玩家感觉是一个游戏。

## 七、变现总览

三条腿,全部不卖强度：

| 变现 | 对象 | 卖什么 | 原则 |
|---|---|---|---|
| 广告 | 大众 | **续用角色技能的机会**(第二次起,有上限)+ 激励/插屏 | 卖机会(用时间换)非强度;玩家主动想要时看,不强制 |
| 抽卡 | 想要能力的人 | 角色（玩法能力）| 靠高分解锁,实力驱动;首抽必出借贷 |
| 皮肤 | 想炫耀的人 | 纯外观 | 绝不带强度,喂 to be somebody |

**红线**：任何变现都不卖数值强度、不破坏公平、不违背「不信廉价成长」。三腿全是「卖机会 / 卖外观」,无一卖永久强度。
