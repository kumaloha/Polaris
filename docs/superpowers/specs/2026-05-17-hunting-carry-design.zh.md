# Hunting（猎场）核心设计 v2：多周期 carry + 可接受性脊柱

状态：经长会话共识锁定的设计主稿。取代 `docs/mvp/09-hunting-core-design.zh.md` 中"单晚漏斗"和"自我侧 4 卡薄版"等过时部分；视觉/界面仍以 `08-hunting-godot-redesign.zh.md` 为方向稿（具体 UI 等外部 HTML 设计稿回来后另定）。命名见 `00-mvp-brief.md`。

## 0. 一句话定位

**它是 Amy 姐培训班的游戏版。** 现实里"读男人、筛选、经营自己"代价极高、还得偷偷摸摸；游戏把它变成低代价、能开天眼、不用对外认账的**实践**——不是教程。机制是培训班的硬核，皮肤是"判断力 + 自我主权"，绝不自称培训班。

玩家的资产不是男人，是**她自己的净值**（社交资本 + 判断力 + 成长）。"选中的爽"只是单笔回报的瞬时 tick；会跨周期复利的 stock 是她的 Net Worth。

## 1. 可接受性脊柱（最高约束，统辖所有文案与反馈）

每个场景的文案/反馈/旁白都必须过这把尺。这是分销和品牌的生死线（首发美国，"钓凯子"=gold digger，污名更重）。

1. **防御框架 > 获取框架。** 永远讲"别被耍 / 看穿红旗 / 别把青春浪费在 False Alpha / 守住精力"，绝不讲"教你钓到他"。
2. **telos = 她自己，不是那个男人。** 赢永远定义为她的净值/判断力/成长；钓到高价值男人只是后果/关键帧，绝不是被陈述的目标。
3. **开天眼 = 招牌机制 + 营销首图。** 最爽、最差异化、道德零负担的幻想。
4. **永不教，只让她发现 + 验证。** 无标准答案；失败便宜；开天眼确认。无答案 = 不是课程 = 是体验。
5. **Amy 姐分散进闺蜜（Nina）。** 女生拒绝付费导师，接受姐妹密语。培训内容由闺蜜群共享智慧给出，毒舌又爱你的口吻。
6. **露骨处上皮肤。** 钱 = 可靠的信号之一（担当），不是目标；烂男人 = 从你身上拿（债主），好男人 = 不让你扛他的成本；道德指针永远指向自我保护，不是 extraction。
7. **分层可接受性。** 表层（看红旗/别被骗/变美/姐妹团）对所有人友好；深层（carry 账本/柠檬市场/镜像债务）通过玩自己浮现。casual 得到"解码男人+自我提升"，硬核在底下挖到全套策略书。

## 2. 资产负债表（三类头寸，每类都能是资产或负债）

**Net Worth = 资产 − 负债**，每周 mark 一次。

**资产侧：**
- **Standing / 社交资本**：跨赛季永久。Correct Read、公开赢、经营好闺蜜 → 涨；Sugar Trap、出丑、债务违约 → 减值。它是柠檬市场逃生梯（决定够得着的男人池质量）。挣来/侵蚀，不是买。
- **Judgment Equity / 判断力**：沉淀为两个看得见的实物——
  - **案底库（Dossier）**：跨赛季永久。每读对一个男人归档；新男人在 First Eye 自动比对（"ping 起来像你烧过的型"），老玩家情报预装、看得更多更早更便宜。
  - **社交权威**：案底厚 → 闺蜜圈当你"读男人的权威"（被咨询的 Nina 角色）→ 解锁更高端的局。
- **Keyframe 收藏**：唯一收藏品，且是挣来的不是抽来的——被验证的未来（主角是她的人生轨迹，男人只作"读对了"的证据，甚至不在画面）。也收录减值坏帧。

> 资产负债表上的资产项是上面三个（Standing / 案底 / Keyframe）。另有两个**耐久自我属性**不单列科目但门控回报：**Control 纪律**（§5，回报质量乘数）与 **Persona**（§8.1，战略身份）。读"自我侧全貌"看这五样。

**负债侧：**
- **Fantasy Debt / 幻想债**：你欠的（拿了糖/局/站台不兑付 → 债主男缠你）+ 你持有的（长期吊着男人当负 carry）。有实感（见 §6）。

**软 ruin：** 净值不归零。欠债 > 资产 → 资不抵债 → 强制摊牌 arc（债主挤满派对节点、爬升冻结，直到 work down）。不是 game over，是 debt workout——那个低频高情绪节点本身。

## 3. 时间结构（已锁）

- **1 Night** = 一次开 App = Ready Room → 一场派对 → 对整本书做决策 → 时间推进。
- **1 Week** = 约 6 Night + 1 结算（mark 净值 / 掉关键帧 / 债务计提）。`[TO-TUNE]`
- **1 Season** = 约 3 Week（约 18 Night）→ 赛季落幕：净值/equity curve 战报卡（可分享），叫"你的这一年，你的决定"。`[TO-TUNE]`
- **跨赛季永久：** 社交层全留——闺蜜关系 + Standing/社交网络 + 案底，连**疤**一起留（赛季末没处理的债务 → 转成 Standing/Control 上的伤带进新赛季）。
- **跨赛季重置：** 男人池、未平男性持仓、Energy、市场。
- 无限模式 = 赛季后解锁，不做首发。

主题：男人是会清算的季度交易；她的网络和判断力永远复利。日留存钩子 = "我的书昨晚变了，今晚得 roll 它"。

## 4. 四个资源（重定义，用大白话）

- **Energy / 精力 = 硬预算。** 每个动作消耗；Night 间部分再生（Solo Reset 加）。"我不能什么都要"的来源。
- **Charm / 魅力 = 短期闸门 + 靠近量。** 装扮/自投/人设决定；决定今晚能进哪档局、多少男人主动靠近、First Eye 信息深度。
- **Position / Standing = 长期闸门 + 稀缺性。** Standing 是跨赛季永久权益（资产负债表上的那项）；"Position"沿用 09 术语，是 Standing 在当前赛季的即时读数。柠檬市场逃生梯。
- **Control / 控制感 = 回报质量乘数，也是"你是不是红旗"的字面机制。** 见 §5。它是数值脊柱。

## 5. Control —— 游戏的核心动词（玩家每次都选）

**Control 不是设计师参数，是玩家每一次交互的实时选择。这就是游戏本身。**

**定义（大白话）：** Control = 这段关系按谁的节奏走。谁等谁、谁为谁改计划、谁在追。不是"你多喜欢他"。

**"让他挣"的具体含义。** 男人能给两种东西：
- **便宜的**（对他零成本，谁都能给）：含糊的"改天约"、深夜"想你了"、夸你、不把话说死。
- **贵的**（对他有成本，只有真在乎才肯给）：确定的时间+地点且他去订、为你改安排、公开承认你、你提的具体要求他真做了、为你拒绝别的选项。

**"让他挣" = 他没拿出贵的之前，你不投入时间精力。** Control 全部的意思 = 在他拿出贵的之前，你忍住不付费。

**它为什么是回报乘数（Adrian A/B 例，写进设计语汇）：** 同一个真·优质男 Adrian——
- **A 路（你让他挣）：** 你只说"定好时间地点告诉我"，他真订（周六八点，具体地方）；他配合你的生活；紧张的是他。开天眼：Correct Read，他净增你身家。
- **B 路（你追了）：** 你"好呀随时!"、等他消息、推掉别的事迁就他。同一个 Adrian，关系按他节奏走。开天眼：False Alpha，控制成本吃掉你；镜像里他从没觉得你特别，你是"随叫随到的备胎"。

**同一个资产，相反结局，差别只在你入场时握的 Control。**

**"你也是别人的红旗"——字面机制：** 低 Control = 你默认在给别人发行幻想债（你是过度投入、等待、迁就、追的那个）。结算时点燃**镜像关键帧**：你看到他对你的 Future Eye，用他口吻带收据——"她随叫随到，围着我转，从没让我挣。糖源，持有成本极低。" 你看见你就是他的 Evan。不是因为你坏，是因为你没握 Control 地玩。同一引擎，两头都能指。

**后果模型 = B（已锁）：** 追的代价**当下就有苗头**，开天眼只是终局放大。**关键防答案键设计：** 苗头必须又真、又能被自己骗过去，且**越追糖越多**——
- 你越追，他**便宜的甜话不减反增**（当下更爽、你更上头）；但他**贵的具体行动悄悄消失**。
- 技能 = 看出"**甜话涨、实事跌**"这个背离，并顶住自己"他只是这周忙"的自我安慰。
- 例：他"宝贝今晚也想你😘"（甜话↑）…… 你"那周末见?" 他"看看哈最近有点乱"（具体↓）。苗头含糊——可能真忙，也可能你已太好得手。读对=技能；自欺=自己挖坑。

**B 带来 A 没有的两样：**
1. **中途能救：** 察觉自己在追 → 收手/Boundary → 真资产回温，柠檬彻底消失（现在就便宜学到，不等开天眼）。Control 是每晚主动管的东西，不是结尾的判决。
2. **开天眼从"唯一老师"变"放大器+终局确认"**；镜像更狠也更公正——"第一周你就看见苗头了，你还是追了"。

**惩罚不说教（过尺）：** 永远防御性 + 可挽回（软 ruin workout）；镜像是发现式教学，Nina 口吻，不是 game over。谨慎玩的 casual 大多拿 Correct Read，很少重触镜像；满口獠牙只咬过度追逐者。惩罚清晰、自招、可逆 = 读起来是深度/正义，不是残忍。

## 6. 认知回路（已锁）

**真相一路上都在迹象里，和现实一样。开天眼 = 确认投资回报的结算 + 给你的读打分，不是"最后才揭真相"的魔法。**

现实从不缺迹象，被烧的女生通常是迹象都在、自己骗自己。游戏 = 低代价练习场，训练读本就在的迹象；开天眼把"几年才知道结果"压缩成即时回报确认。

**硬规则：会读的玩家在打开开天眼前，就应能凭累积迹象叫出结果。**

迹象来源链：
1. **First Eye（投前，便宜，部分）**：表面信号 + 聊天证据 + 案底自动打标。你看到的是**说法**，不是真相。Charm/Evidence Study/Trade Receipts 买**深度**（多一条线索），永不买确定性。故意定价错误住在这里：表面≠终值。
2. **派对（活证据）**：他对你探测的反应（tell）把"说法"转成"读"。可带着不确定离场（那是你扛的定价风险）。
3. **持仓期（B 的早期退化）**：甜话涨/实事跌的背离 + 谁在追谁。
4. **你行动（Date/Observe/Test/Cut）**：基于以上的读下注。
5. **Future Eye（结算）**：确认回报 + 打分。"叫对了——10 年回报放大给你看" 或 "叫错了——迹象一直在（镜像）"。

**迹象永远充分。** 伪装拨盘/案底改的是"要多少功力才读得出"，不是"读不读得出"。难度涨，可能性永远在——这才叫"和现实一样"。

**开天眼对不同玩家：** 高手 = 回报量级（10 年复利铺开，判断被兑现的爽）+ 镜像；新手 = 严师（迹象都在你没读出，Nina 口吻、可挽回、靠发现）。惊讶留新手，兑现感留高手。

## 7. 多周期循环（一个 Night 具体长这样）

1. **Ready Room**：先看昨晚你这本书的变化（持仓变异、债主冒头、闺蜜脸色）→ 投资自己 → 定人设/今晚意图。
2. **闺蜜闸门 + 派对地图**：今晚谁带你（=派对档=男人池质量），由你和各闺蜜的信用线状态决定；可选军师读（帮闺蜜挑男人）。
3. **First Eye**：今晚男人的部分情报，被案底折扣。
4. **Party Encounter**：读牌循环。
5. **After-Party Phone**：对**整本书**操作（不只今晚那个；旧持仓、债主也在这里逼你动）。
6. **时间推进**：持仓按各自 carry/凸性变异；幻想债计提；Energy 部分再生。
7. 每约 6 Night：**结算**（mark + 关键帧 + 镜像若资不抵债）。

## 8. 六个场景（重设计）

### 8.1 Ready Room / 镜前准备（自我侧 = 机会成本）

4 张卡不变（Beauty Care / Work Win / Solo Reset / Evidence Study），深度来自**机会成本**——一个 Night 自投行动有限，做不全：
- Work Win → 长期 Standing/access↑，但今晚靠近你的男人变少（短期流动性↓）。
- Beauty Care → 今晚 Charm/靠近↑（短期），不沉淀耐久权益。
- Solo Reset → Energy↑（今晚能多动几轮/多扛持仓），今晚不涨 Charm/Standing。
- Evidence Study → First Eye 深度↑（信息优势），今晚零存在感。

正确分配取决于今晚书的状态（债重夜→Solo Reset 才扛得起 workout；爬升夜→Work Win；高风险读牌夜→Evidence Study）。先看书的变化，再分配。不加新系统。

**Persona = 战略身份**（非每晚开关）：世界会记住的声誉；切换有代价（扰乱市场、暂时 Standing 微跌）；每个 Persona 改变哪些读牌线生效（Power Darling 让 Boundary 更重但触发低自尊男防御；Soft Sun 拓宽靠近但招更多负 carry；Rare Girl 抬闸但惩罚过度 Engage）。Persona = "你选择成为谁"，不是钓鱼戏服。

### 8.2 Girlfriend Night + Party Map / 闺蜜局 + 派对地图

**闺蜜 = 一条循环信用额度（你拿杠杆的渠道）。** Position 基本上就是你三段闺蜜关系的状态，有脸。
- 经营好 → 额度扩张（多带你、当众背书、给更好情报）。
- 经营差 → 抽贷（不叫你了，派对档掉回残羹）或负面 reference（Standing 计提）。
- 情绪锋利点：信用紧缩往往发生在你最需要的那晚（被债主逼到要 workout 时，关系紧张的闺蜜偏偏不带你）。

**军师子模块（帮闺蜜挑男人）：** 闺蜜把她在考虑的男人拿给你，你给放行/警告/否，多周期结算。读对 → 她信任复利（带你进更好的局、护你）。读错或私心误导 → 友谊计提甚至凉。**暗机制：** 故意误导闺蜜搞掉竞争对手 = 你向社交图发行幻想债；被 Nina 型看穿 → Standing 崩（女人之间的"你也是红旗"）。**限速器：** 参谋以你信誉为抵押——给太多读/给错一个，你的话贬值，军师收益自衰减。它是不烧自己 Energy 的高频读牌面（接 brief 高频/低频拆分）。

### 8.3 First Eye / 第一天眼

只看资料不预测未来。展示：男人照片、身份标签、Energy Cost、Risk、Opportunity、近期聊天记录、**案底自动打标**。Charm/Evidence Study/Trade Receipts 买信息深度（多一条线索），不买确定性。故意定价错误：Evan 表面最优实则负 carry；Leo 表面最差实则凸性赢家；Adrian 表面强实则控制成本陷阱（除非派对里 Boundary 打对）。玩家的 alpha = 这个 spread。选择：Set Primary / Set Backup / Enter。

### 8.4 Party Encounter / 派对闯关（最高频，读牌对局）

**不是答题，是带不完全信息的读牌对局。** 他有两条隐藏轨（你看不到数字）：
- **Interest**（他多想要你）、**Respect**（他是否当你高价值、会不会兑现成具体行动）。

终值 = 派对结束时两轨组合 × 他隐藏 Type（资源/糖/成长 + 案底后的伪装变体）。高 Interest+低 Respect→Sugar Trap 轨；够 Interest+高 Respect→Correct Read 轨；Interest 太低→仓位不开（有时这恰是对的——读出柠檬没浪费 Energy）。

四个动作 = 互相对冲的取舍，效果取决于他的隐藏 Type（这就是读）：

| 动作 | 移动 | 成本 | 关键 |
|---|---|---|---|
| Engage | Interest↑ Respect↓(小) | Energy 1 `[TO-TUNE]` | 廉价多巴胺；滥用→你成他的易得选项=你是红旗那条线 |
| Boundary | Respect↑；太早/对低Interest男→Interest↓他撤 | Energy 1 | 技术动作；对资源男well-timed→转化；对Sugar男→他兑现不了→他的闪躲=tell，白拿情报且正确没投 |
| Social Proof | Respect↑ 他压力↑ | 1 Attention（每场约2）`[TO-TUNE]` | 强，但对自尊敏感型(成长)反噬——他撤、Interest崩；用前得先读出他扛不扛得住 |
| Exit | 锁定当前轨、结束 | ≈0 | 读出柠檬就早退=正确防守；对真资产太早退=Missed Growth 的种 |

**读牌循环（杀掉答题感）：** 每轮 你出动作 → 他的反应台词+体态变化（tell）→ 更新你的读 → 下轮适配。例 tell：你 Boundary → 他给具体=资源/Correct；转暖又含糊=Sugar；刺一下又收回认真=成长。你 Social Proof → 竞争跟上=安全(资源)；蔫了消失=自尊敏感(成长,以后别SP他)；词更甜行动没变=Sugar。**你提取的情报本身就是资产**（就算这仓不开，也进案底）。

**反馈不走数值条（违背 08 §5.6 的 Interest Meter，故意）：** 只走他的台词+表情/体态 + 一句赛后毒舌读总结。meter = 表格感 = Codex 那版死因。`[待 HTML 视觉稿回来后定具体呈现]`

旋钮 `[TO-TUNE]`：一场盯几个男人（简单版 1 个；进阶 Primary+Backup 同场分配）；轮数（6 偏长，试 4-5，高频屏要快）；Attention 资源留不留（不留则 Social Proof 只烧 Energy，更轻）。

### 8.5 After Party Phone / 派对后手机（对整本书操作）

不只今晚那个男人。这里同时处理：今晚目标的 Date/Observe/Test/Cut + 旧持仓的演化（观察仓升级或衰减）+ **债主男缠你**（他每个 Night 占一个 Energy 槽，当众砸你 Standing，借钱/加码无理要求，不接受 Cut）。清债两条路：还（强制处理 arc，真 Energy/Control）或违约（Standing 计提，equity curve 留疤）。闺蜜群吐槽 = Nina 口吻（可接受版 Amy 姐）。

选择：Date（主投资，只一个，Energy 高消耗）/ Observe（低成本持仓）/ Test（给明确验证任务，Energy 中）/ Cut（保护精力）。

### 8.6 Future Eye / 结算（回报确认 + 打分 + 镜像）

触发：只在玩家 Date/Observe/Test 之后；每周结算。未来关键帧（不做表格）：3 周 / 6 个月 / 3 年 / 10 年。结果类型：Correct Read / Sugar Trap / Slow Upside / False Alpha / Missed Growth。确定（给定 男人型 × 你的选择 × Control × 序列质量），非 RNG = 挣来的全知。

Keyframe 放被验证的未来（她的轨迹，男人作证据，可不在画面），含减值坏帧。镜像帧在资不抵债时全开。赛季落幕出净值/equity curve 战报卡。
