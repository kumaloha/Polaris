# 猎场 · 鉴渣小游戏 设计（休闲 · 最终 · 替代旧复杂方案）

日期：2026-05-18
定位：**休闲小游戏**。这份 spec 是当前权威游戏设计，**取代**旧的复杂 party 方案（两轴好感度 / Control 追挣 / 分支对话 / 判断力权益 / 资产负债表 / 软 ruin / 赛季 / hub 外壳改造——全部归档搁置、不删、不再推进）。已做好并上线的资产（引擎核心、`Content.men()` 12 人语料、`PeekChat`、UiKit/Theme/Loc、Peek 屏）继续复用，本设计踩在它们上面。

soul 不变：冷高级、反乙女、她在上位、纯代码无美术。

---

## 一句话

挑男人识渣男的解压小游戏。**先开天眼看他对别人怎么说（他的底）→ 他当面对你演甜话 → 你辨「好男人/渣男」并选个态度（拆穿/试探/走开）→ 一句冷台词结局。** 一局 20–40 秒。

## 一回合

1. **开天眼（先看，必看）**：甩出他对别人说的**最狠 1–2 句**（从已写好的 `others_chat` 里抽最扎的；不渲染整段）。这是他的底，也是你下一步唯一的依据——不看就没法判，"懒得看"被结构性消灭。
2. **当面**：他对你说一句他的表面话（已写好的 `chat`，甜/装/油，专门勾你误判）。
3. **辨 + 选**：玩家判断他是 **好男人 / 渣男**，并选一个态度——固定 3 个：**拆穿 / 试探 / 走开**（不是对话树，就是 3 个按钮）。
4. **结局**：一句冷台词收尾，由 **(真相 × 选择)** 决定（见下）。读对又出手对 = 爽（当面拆穿渣男 / 干净走人）；误伤真男人或被渣男骗 = 冷回旋镖（你失了好的 / 他玩了你）。
5. **计数**：底部一句"看穿 X / N"或连对 streak。**无经济、无两轴、无 Control、无资产负债表、无赛季、无 hub 外壳**。

## 好 / 渣 判定（零新建模型，复用 `hidden_type`）

- `high_sugar` → **渣男**
- `resource` / `growth` → **好男人**

12 个人天然难易混合：伪装的（表面≠真相）是难关和爽点，表里一致的是简单关/校准。

## 结局矩阵（控内容量：模板，不逐人手写）

按 **(他是好/渣) × (拆穿/试探/走开)** 给约 6 句冷台词模板，套他名字即可。要点：

- 渣 + 拆穿 = 最爽（当面戳穿、他破功、你走）。
- 渣 + 走开 = 干净利落的胜。
- 渣 + 试探 = 他继续演，你看得更清（小赢/警示）。
- 好 + 拆穿 = 回旋镖（你误伤了一个真的，他沉默走开，你亏）。
- 好 + 走开 = 你错过了（淡淡的可惜）。
- 好 + 试探 = 中性偏好（他经得起，你看到真的）。

不做 12×N 的逐人结局；语料厚度留在"他对别人那句"上，结局走模板。

## 复用 / 新增 / 改造

- **复用**：`Godot/core/Content.gd` `men()`（`chat`=他对你、`others_chat`=他对别人、`hidden_type`=真相）、`Godot/core/PeekChat.gd`、`Godot/ui/UiKit.gd`/`Theme.gd`/`Loc.gd`。
- **新增（极少）**：好/渣映射（一个纯函数，`hidden_type`→bool，绝不在 UI 暴露 `hidden_type` 本身）；结局矩阵（~6 模板，Loc 追加）；回合状态机。
- **改造**：把现在的 `Godot/ui/Peek.gd`（两栏并列那版）**原地改造**成这个回合制小游戏（intel → 当面 → 辨+选 → 结局 → 下一局 + 计数）。**就一个简单屏，不并行开新场景。** `Peek.tscn` 保留。
- 抽"最狠一句"：从 `others_chat` 取第 1 条（或一个确定性挑选，无 RNG，符合项目确定性原则）；想看全的可留一个可选"看完整"，但**核心循环只要那一句**。

## 明确不做（范围外）

两轴好感度、Control 追/挣、分支对话树、判断力权益结算、资产负债表、软 ruin、赛季流转、hub 外壳（常驻女主+底栏弹层）、开天眼稀缺/体力成本、约会侧"看十年后"、男性语料量产（12 人够休闲版用，要扩是后续可选）。引擎其余系统不动、`tuning.json`/`play.gd`/`ui_smoke.gd`/hub 不碰；headless 测试全绿。

本 spec 只锁"这个休闲小游戏是什么、怎么一回合"。短、简单、踩在已做好的零件上。
